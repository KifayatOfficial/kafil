## COMMON QUESTIONS

**Q: Can I skip the legal stuff?**
A: No. Spend 2 hours reading the Pakistan compliance section. Saves headaches later.

**Q: Do I need to hire before launching?**
A: No. You can build MVP solo. Hire Month 4 if it's working.

**Q: What if timeline slips?**
A: Expect 1-2 weeks of slippage. Adjust features, not launch date.

**Q: Should I fundraise?**
A: Not needed for MVP. Only if you want to scale faster (Month 4+).

**Q: What if competitors copy?**
A: They'll copy features, not network effects. Move fast, build community, win.

**Q: How do I know if it's working?**
A: Check weekly metrics against targets in Execution Map. Red flags = pivot.
